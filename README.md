# Literary Text Analysis Tool

## Description
A Java command-line tool that analyzes literary texts and compares similarity between two documents.

## Features
- Word frequency analysis with stopword filtering
- Vocabulary richness calculation
- Sentiment analysis (Positive / Negative / Neutral)
- Top 20 most frequent words using bubble sort
- Cosine similarity score between two text files
- Exports full analysis report to .txt file

## Concepts Used
- Object-Oriented Programming (Abstract class, Interface, Inheritance)
- File I/O
- Cosine Similarity Algorithm
- Data Structures (arrays, frequency counting)

## How to Run
1. Compile: javac TextAnalysis.java
2. Run: java TextAnalysis
3. Provide paths to two .txt files when prompted
4. Reports saved as analysis_report_1.txt and analysis_report_2.txt
